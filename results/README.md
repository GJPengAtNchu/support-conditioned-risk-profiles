# Results directory

Generated CSV, JSON, table, and cache files are not tracked in this release
repository. They can be regenerated by running the experiment scripts or
downloaded from:

[Google Drive data and cached results](https://drive.google.com/drive/folders/1degj8NKU1FJTib9hocOrUMixivfM_5vt?usp=sharing)

By default, scripts write outputs under `outputs/`.
